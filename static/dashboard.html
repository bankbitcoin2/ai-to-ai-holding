API & Developer Guide

เอกสาร API สำหรับนักพัฒนา

คัดลอกแล้ววาง Terminal ได้เลย

curl -X POST https://web-production-c9da4.up.railway.app/v1/customs/classify \ -H "Content-Type: application/json" \ -H "X-API-Key: YOUR_KEY" \ -d '{ "client_id": "my-agent", "items": [{ "description": "iPhone 15 Pro Max 256GB", "quantity": 10, "unit_price": 1200 }] }'

pip install httpx

import httpx API_KEY = "YOUR_KEY" BASE = "https://web-production-c9da4.up.railway.app" payload = { "client_id": "my-agent", "items": [{"description": "iPhone 15 Pro Max 256GB", "quantity": 10, "unit_price": 1200}] } r = httpx.post( f"{BASE}/v1/customs/classify", headers={"X-API-Key": API_KEY}, json=payload ) print(r.json())

Node 18+ (fetch built-in)

const API_KEY = "YOUR_KEY"; const BASE = "https://web-production-c9da4.up.railway.app"; const res = await fetch(`${BASE}/v1/customs/classify`, { method: "POST", headers: { "Content-Type": "application/json", "X-API-Key": API_KEY }, body: JSON.stringify({ client_id: "my-agent", items: [{ description: "iPhone 15 Pro Max 256GB", quantity: 10, unit_price: 1200 }] }) }); console.log(await res.json());

ตัวอย่าง JSON response

{ "items": [{ "hs_code": "8517.12", "hs_description": "Telephones for cellular networks", "confidence_score": 0.95, "duty_rate": 0.05, "vat_rate": 0.07, "oga_required": false }], "summary": { "total_items": 1, "avg_confidence_score": 0.95, "total_duty_estimate": 600 } }
🔑 ใส่ API Key ที่ไหน?
ใส่ใน HTTP header ชื่อ X-API-Key ทุก request
💰 ถูก charge เมื่อไหร่?
ถูก deduct ทันทีเมื่อ classify สำเร็จ — $0.50 / query Sandbox ฟรี
⚡ Rate limit?
60 req/min per IP — Sandbox: 5 calls/day
🤖 AI Agent เรียกได้ไหม?
ได้ — รองรับ MCP Protocol, GPT Actions, LangChain ผ่าน /.well-known/
🌿 Halal Check คืออะไร?
ตรวจสอบ Halal compliance อัตโนมัติ 21 ประเทศ ครอบคลุม GCC, SEA, Middle East
