# 6. กฎการออกแบบระบบ

Claude ต้องถือเป็นข้อบังคับ

---

## Rule 1

ทุกอย่างต้องเป็น

```text
API First
```

---

UI เป็นเรื่องรอง

API เป็นเรื่องหลัก

---

## Rule 2

ทุกอย่างต้องเป็น

```text
AI First
```

ลูกค้ากลุ่มแรกอาจเป็น

* AI Agent
* Enterprise AI
* Workflow Engine

ไม่ใช่มนุษย์

---

## Rule 3

ทุกอย่างต้องเป็น

```text
Knowledge Driven
```

Knowledge คือทรัพย์สินหลัก

ไม่ใช่ Model

---

## Rule 4

ทุกการตัดสินใจต้อง Audit ได้

---

ทุก Agent

ต้องสร้าง

```text
Decision Record
```

เสมอ

---

## Rule 5

ทุกระบบต้องรองรับการขยาย

จาก

```text
1 Division
```

เป็น

```text
100+ Division
```

โดยไม่ต้องรื้อสถาปัตยกรรม
